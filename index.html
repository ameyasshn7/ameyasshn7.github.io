Dota 2 Draft Recommendation System

The Dota 2 Draft Recommendation System employs machine learning to suggest three suitable heroes for the last pick during the drafting phase of the game. It considers the user's draft, the opponent's draft, and banned heroes. Data is acquired from OpenDota and preprocessed, leading to the training of deep learning models like CBOW and LSTM. A web application facilitates the drafting process. The model provides commendable accuracy, although 'Mid' heroes are often suggested as last picks. The system adapts to Dota 2's ever-changing patches and offers valuable assistance to players in making informed hero selections, enhancing gameplay and strategy development.

Diabetic Retinopathy Detection

Diabetic retinopathy, a diabetes-induced eye condition, can lead to blindness if not detected early. Python's image processing capabilities enable automated analysis of retinal images, identifying signs like microaneurysms. Early diagnosis aids timely intervention, preventing vision loss. Python's efficiency with large datasets makes it invaluable in managing diabetic retinopathy at scale, \ improving patient outcomes, and easing healthcare systems. The repository houses a deep learning project for classifying retinal images into five classes, using DenseNet121 as the model. It includes data preprocessing, augmentation, training, evaluation, and test prediction. The model achieves a 72.73% accuracy, showcasing its diagnostic potential. Data source: Kaggle Diabetic Retinopathy Detection Dataset.
